# 通用资源采集助手

一款面向 Windows 的开源桌面工具，用于读取公开网页，发现文章详情页，并整理文章标题、分类和公开网盘分享链接。

本工具针对 **藏知库** 的资源整理与批量导入流程进行了适配，同时也可以完全独立使用，并支持为不同网站保存自定义采集规则。

> 本项目只提供网页信息整理能力，不包含网盘登录、自动转存、验证码绕过或访问限制绕过功能。请仅采集你有权访问、保存和分享的内容，并遵守网站条款与当地法律。

## 功能

- 输入网站首页、栏目页或列表页进行采集
- 自动发现分页和文章详情页
- 内置网页点选器，为陌生网站创建采集规则
- 支持极速、稳定断点和自定义速度模式
- 识别迅雷、夸克、百度、阿里云盘、UC、115、天翼云盘链接
- 支持全部网盘、迅雷优先和指定网盘模式
- 自动归类电影、电视剧、短剧、纪录片、动漫动画、综艺、读物、音频、学习和游戏
- 支持暂停、继续、失败重试和 Excel 总表导出
- 迅雷链接可单独导出为 TXT，供其他独立工具使用

## 已适配网站

- UVWHD：保留旧版专用解析规则
- dyyjv.com：支持分页、详情页和多网盘链接
- 其他公开网站：先自动识别，必要时使用内置网页点选器建立规则

通用识别无法保证适配所有网站。遇到登录、验证码、Cloudflare 真人验证或访问限制时，程序应暂停，不会尝试绕过。

## 开发运行

普通用户可以在仓库的 [Releases](../../releases) 页面下载 Windows 便携版，无需安装开发环境。

### 从源代码运行

需要 Node.js 20 或更高版本，并使用 pnpm 安装依赖。

```powershell
pnpm install
pnpm test
pnpm start
```

构建 Windows 便携版：

```powershell
pnpm run dist
```

## 数据与隐私

采集进度和自定义网站规则保存在本机应用数据目录，不提交到代码仓库。仓库也不包含 Cookie、账号凭据、采集结果或网盘资源清单。
